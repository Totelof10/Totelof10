# Deploiement Frontend (Mode Demo)

Ce dossier contient une copie du frontend runnable sans backend pour une demo portfolio.

## Lancer la demo

1. Installer les dependances:

```bash
npm install
```

2. Demarrer en local:

```bash
npm run dev
```

3. Ouvrir l'URL Vite affichee dans le terminal.

## Compte de demo

- Username: `demo`
- Password: `demo123`

Le mode demo est active via `VITE_DEMO_MODE=true` dans `.env`.
Les donnees sont stockees localement dans le navigateur (`localStorage`) pour conserver les changements pendant la session de demo.
